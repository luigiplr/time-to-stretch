## Requirements
- PC running Windows 7, OS X 10.9, Ubuntu 14.x or above.

<br>

## Contributing
### Global Requirements
- `gulp`
- `node & npm`

### Getting Started
- `npm install`

To run the app in development:
- `gulp start-dev`

### Building & Release
- `gulp release`
